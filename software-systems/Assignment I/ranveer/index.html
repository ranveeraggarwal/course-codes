My Academic Projects

Chess Titans

Guide: Prof. Amitabha Sanyal

Developed a one player chess game, in PLT Scheme using the DrRacket in-built GUI Toolkit, using functional programming.
The game's AI makes use of the Minimax algorithm with alpha-beta pruning and heuristics to decide the best move for the computer, at any stage of the game.

E-Ball

Guide: Prof. Abhiram Ranade

The classic brick-breaker game,developed in C++ using simplecpp graphics library. Used principles of elementary particle dynamics to give directions to the ball.
The game generates random levels with varying difficulty and ball speeds.

ATM Modelling

Guide: Prof. Ganesh Ramakrishnan

Predicted optimal refilling time of 5 ATMs in different areas of a city at various times using various probability distribution models.

Hacks and Websites

Android Desktop Controller

Intstitute Technical Summer Project (ITSP) 2013

Android Desktop Controller is an Android application that uses Bluetooth to perform various tasks on your PC. The application has two modules - Android client application and the PC server application. The PC application runs in background while the Android app performs the desired tasks.
To make this application, Java's Bluecove library for Bluetooth and Robot library for assigning key functions were extensively used.

Rumor Roll!

Yahoo! HackU 2013

Want to hear rumors about which phone has leaked in the market? Or which movie hero is in the news these days? Behold Rumor Roll! A web application that gives you all the rumors related to the given search query.
This application used Yahoo! Boss API, YQL, Twitter API v2.0, YouTube API and Twitter Bootstrap v3.0.
